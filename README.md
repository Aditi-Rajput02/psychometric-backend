# Psychometric Assessment Platform – Backend

Backend API for the **Psychometric Assessment Platform**.
This server handles **user authentication, assessment management, result storage, and user progress tracking**.

Built using **Node.js, Express, and MongoDB**.

---

## 🚀 Features

* User registration and authentication
* Role-based access (Student / Admin)
* Psychometric assessments management
* Assessment progress tracking
* Store and retrieve user results
* Secure password handling
* RESTful API architecture

---

## 🛠 Tech Stack

* **Node.js**
* **Express.js**
* **MongoDB**
* **Mongoose**
* **JWT Authentication**
* **bcrypt**
* **dotenv**

---


## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Aditi-Rajput02/psychometric-backend
```

Navigate to the backend folder:

```
cd backend
```

Install dependencies:

```
npm install
```

---

## 🔑 Environment Variables

Create a **.env** file in the root directory.

Example:

```
PORT=5000
MONGO_URI=mongodb://localhost:27017/psychometric
JWT_SECRET=your_jwt_secret
```

---

## ▶️ Running the Server

Start the development server:

```
npm run dev
```

or

```
npm start
```

Server will run on:

```
http://localhost:5000
```
